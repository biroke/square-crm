Procédure d'installation :

Il faut installer Java(TM) Cryptography Extension (JCE) Unlimited Strength Jurisdiction Policy Files 6 sur la JVM pour permettre l'encryptage AES des mots de passe de l'espace client.

- Récupérer l'archive jce_policy-6.zip située dans le répertoire installation ou la télécharger via l'url : 
https://cds.sun.com/is-bin/INTERSHOP.enfinity/WFS/CDS-CDS_Developer-Site/en_US/-/USD/ViewProductDetail-Start?ProductRef=jce_policy-6-oth-JPR@CDS-CDS_Developer
ou pour la version 5 via l'url :
https://cds.sun.com/is-bin/INTERSHOP.enfinity/WFS/CDS-CDS_Developer-Site/en_US/-/USD/ViewProductDetail-Start?ProductRef=jce_policy-1.5.0-oth-JPR@CDS-CDS_Developer
- Décompresser l'archive
- Suivre les instructions du fichier README.txt extrait avec l'archive qui consiste simplement à remplacer des fichiers de la JVM.
